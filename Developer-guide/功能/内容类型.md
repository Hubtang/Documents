# 内容类型

*content\_type*元素列出了可用的内容类型。

             
    <capabilities>
    <version major="3" minor="3">
    ...
    <content_types>
    <content_type>text</content_type>
    <content_type>binary</content_type>
    </content_types>
    ...
    </version>
    </capabilities>
