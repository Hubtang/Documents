# 主机网络接口卡

*host\_nic\_states*元素罗列出了主机网络接口卡可能处于的状态。

             
    <capabilities>
    <version major="3" minor="3">
    ...
    <host_nic_states>
    <host_nic_state>down</host_nic_state>
    <host_nic_state>up</host_nic_state>
    </host_nic_states>
    ...
    </version>
    </capabilities>
