# 磁盘接口

*disk\_interfaces*元素罗列出了使用虚拟磁盘的仿真接口协议选项。*disk\_interface*选项

             
    <capabilities>
        <version major="3" minor="3">
            ...
            <disk_interfaces>
               <disk_interface>ide</disk_interface>
               <disk_interface>virtio_scsi</disk_interface>
               <disk_interface>virtio</disk_interface>
            </disk_interfaces>
            ...
        </version>
     </capabilities>
