# 电源管理代理类型

*pm\_proxy\_types*元素罗列出了电源管理代理类型。

             
    <capabilities>
    <version major="3" minor="3">
    ...
    <pm_proxy_types>
    <pm_proxy_type>cluster</pm_proxy_type>
    <pm_proxy_type>dc</pm_proxy_type>
    </pm_proxy_types>
    ...
    </version>
    </capabilities>
