# 快照状态

*snapshot\_stutuses*元素列出了可有的快照的状态。

             
    <capabilities>
    <version major="3" minor="3">
    ...
    <snapshot_statuses>
    <snapshot_status>ok</snapshot_status>
    <snapshot_status>locked</snapshot_status>
    <snapshot_status>in_preview</snapshot_status>
    </snapshot_statuses>
    ...
    </version>
    </capabilities>
