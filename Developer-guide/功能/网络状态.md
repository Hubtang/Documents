# 网络状态

*network\_states*元素罗列出了网络可能处于的可用的状态。

             
    <capabilities>
        <version major="3" minor="3">
            ...
            <network_states>
                <host_state>operational</host_state>
                <host_state>non-operational</host_state>
            </network_states>
            ...
        </version>
    </capabilities></host_state></host_state></network_states></version></capabilities>
