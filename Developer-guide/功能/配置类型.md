# 配置类型

*configuration\_types*元素罗列出了可用的配置类型。

             
    <capabilities>
        <version major="3" minor="3">
            ...
            <configuration_types>
            <configuration_type>ovf</configuration_type>
            </configuration_types>
            ...
        </version>
    </capabilities></configuration_type></configuration_types></version></capabilities>
