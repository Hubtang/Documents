# 模板状态

*template\_states*元素罗列出了模板可能处于的状态。

             
    <capabilities>
        <version major="3" minor="3">
            ...
            <template_states>
               <template_state>illegal</template_state>
               <template_state>locked</template_state>
               <template_state>ok</template_state>
            </template_states>
            ...
        </version>
    </capabilities>
