# 处理器模式

*cpu\_modes*元素罗列出了可用的CPU模式。

             
    <capabilities>
    <version major="3" minor="3">
    ...
    <cpu_modes>
    <cpu_mode>custom</cpu_mode>
    <cpu_mode>host_model</cpu_mode>
    <cpu_mode>host_passthrough</cpu_mode>
    </cpu_modes>
    ...
    </version>
    </capabilities>
