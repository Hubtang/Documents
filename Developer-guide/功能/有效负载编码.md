# 有效负载编码

*payload\_encodings*元素罗列出了可用于负载的不同编码。

             
    <capabilities>
    <version major="3" minor="3">
    ...
    <payload_encodings>
    <payload_encodings>base64</payload_encodings>
    <payload_encodings>plaintext</payload_encodings>
    </payload_encodings>
    ...
    </version>
    </capabilities>
