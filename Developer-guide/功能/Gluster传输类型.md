# Gluster传输类型

*transport\_types*元素罗列出了Gluster卷可有的传输类型。

             
    <capabilities>
    <version major="3" minor="3">
    ...
    <transport_types>
    <transport_type>tcp</transport_type>
    <transport_type>rdma</transport_type>
    </transport_types>
    ...
    </version>
    </capabilities>
