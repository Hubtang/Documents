# 报告设备类型

*reported\_device\_types*元素列出了可用的报告设备类型。

             
    <capabilities>
    <version major="3" minor="3">
    ...
    <reported_device_types>
    <reported_device_type>network</reported_device_type>
    </reported_device_types>
    ...
    </version>
    </capabilities>
