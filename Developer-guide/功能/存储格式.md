# 存储格式

*存储格式*元素为存储元数据罗列出了可用 *format*版本

             
    <capabilities>
        <version major="3" minor="3">
            ...
            <storage_formats>
                <format>v1</format>
                <format>v2</format>
                <format>v3</format>
            </storage_formats>
            ...
            </version>
    </capabilities>
