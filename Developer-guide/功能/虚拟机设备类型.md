# 虚拟机设备类型

*vm\_device\_types*元素罗列出了的虚拟机可能处于的状态。

             
    <capabilities>
    <version major="3" minor="3">
    ...
    <vm_device_types>
    <vm_device_types>floppy</vm_device_types>
    <vm_device_types>cdrom</vm_device_types>
    </vm_device_types>
    ...
    </version>
    </capabilities>
