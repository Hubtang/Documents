# 步骤类型

*step\_types*元素罗列出了不同的步骤类型。

             
    <capabilities>
    <version major="3" minor="3">
    ...
    <step_types>
    <step_type>validating</step_type>
    <step_type>executing</step_type>
    <step_type>finalizing</step_type>
    <step_type>unknown</step_type>
    </step_types>
    ...
    </version>
    </capabilities>
