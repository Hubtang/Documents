# 看门狗操作

*watchdog\_actions*元素罗列出了当计时器达到零时，看门狗可能执行的操作。

             
    <capabilities>
    <version major="3" minor="3">
    ...
    <watchdog_actions>
    <action>none</action>
    <action>reset</action>
    <action>poweroff</action>
    <action>pause</action>
    <action>dump</action>
    </watchdog_actions>
    ...
    </version>
    </capabilities>
