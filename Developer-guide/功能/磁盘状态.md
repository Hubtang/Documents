# 磁盘状态

*disk\_states*元素罗列出了磁盘可能处于的状态。

             
    <capabilities>
    <version major="3" minor="3">
    ...
    <disk_states>
    <disk_state>illegal</disk_state>
    <disk_state>locked</disk_state>
    <disk_state>ok</disk_state>
    </disk_states>
    ...
    </version>
    </capabilities>
