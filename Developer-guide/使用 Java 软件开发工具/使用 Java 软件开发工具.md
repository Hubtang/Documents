# 使用 Java 软件开发工具

本章概述了几个怎么在 Java IDE 下使用 Java 软件开发工具的大致步骤。需要在
Java IDE 下使用 Java 软件开发工具的话，*ovirt-engine-sdk-java* 和
*ovirt-engine-sdk-java-javadoc* 中的 .jar 文件必须以外部库的方式加入到
Java 工程中。如下的 .jar 文件也需要加入：

-   *apache-commons-logging*

-   *commons-beanutils*

-   *commons-codec*

-   *httpclient*

-   *httpcore*
