# 为虚拟机添加一个虚拟磁盘

**概述**<br/>
下面介绍了把一个虚拟磁盘添加到一个虚拟机的方法。当虚拟磁盘被添加到一个虚拟机后，这个虚拟机就可以访问这个磁盘了。

**为虚拟机添加一个虚拟磁盘**
1. 点**虚拟机**标签页并选择一个虚拟机。

2. 在详情框中选**磁盘**页。

3. 在详情框的上部菜单中点**添加**。

4. 在**大小（GB）** 项中输入磁盘的大小（以 GB 为单位）。

5. 在**别名**项中输入别名。

6. 点**添加虚拟磁盘**窗口右下角的**确定**。

您所添加的磁盘会出现在详情框中。

**结果**<br/>
虚拟磁盘被添加到虚拟机上。

> **注意**
>
> 为虚拟机添加或删除磁盘不会消耗“配额（Quota）”资源。

> **注意**
>
> 使用以上方法可以把一个虚拟磁盘添加到多个虚拟机上。
