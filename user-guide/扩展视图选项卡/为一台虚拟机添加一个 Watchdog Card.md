# 为一台虚拟机添加一个 Watchdog Card

**概述**<br/>
为虚拟机添加一个 Watchdog 卡

**为虚拟机添加一个 Watchdog 卡**

1. 点**虚拟机**页并选择一个虚拟机。

2. 点**编辑**键打开**编辑虚拟机**窗口。

3. 点**显示高级选项**显示所有标签页，再选择**高可用性**标签页。

4. 从 **Watchdog 型号**下拉菜单中选择 watchdog 的型号。

5. 在 **Watchdog 操作**下拉菜单中选择一个操作。这个操作会在 watchdog 事件发生时在虚拟机上执行。

6. 点**确定**。

**结果**<br/>
您为虚拟机添加了一个 watchdog。
